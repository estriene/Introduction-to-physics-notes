简要浏览课本内容

```mermaid
timeline
title 力学领域物理学专家
section 阿基米德
    简介 : 前1世纪古希腊哲学家、数学家、<br>物理学家、发明家、工程师、<br>天文学家
    贡献 : 建立静力学平衡定律，<br>最早发现阿基米德浮力定律、<br>滑轮原理和杠杆原理
    著作 : 《浮体》是流体静学的第一部专著，<br>《论平面平衡》-书中建立了<br>静力学平衡的基础
section 伽利略
    简介 : 16世纪意大利天文学家、哲学家、数学家和物理学家
    贡献 : 用自制的望远镜观测到的金星"相位"现象，<br>为"日心说"提供了决定性的实验证据，<br>首个通过人工设计实验及思想推理获得自由落体定律和惯性定律
    著作 : 《关于托勒密和哥白尼的两大世界体系的对话》，<br>《关于力学和运动的两门新科学的对话》创立了运动学和动力学的基本原理和方法
section 牛顿
    简介 : 17世纪英国数学家、物理学家和天文学家，<br>经典力学的创建者。
    贡献 : 牛顿运动定律和万有引力定律<br>四个定律奠定了牛顿力学的基础，<br>使力学有了精练完美的表达，成为系统完整的科学
    著作 : 《自然哲学的数学原理》<br>一书阐述了牛顿所研究的重要成果，<br>即牛顿运动定律和万有引力定律
```

#### 历史

简要浏览课本内容

```mermaid
timeline
title 热学领域物理学专家
section 摄尔修斯
    简介 : 18世纪瑞典天文学家，提出摄氏温标
    贡献 : 给出水的沸点与大气压的关系曲线，创立摄氏温标
    著作 : 《温度计两个确定刻度的观测》，<br>提出了水的凝固点与纬度和大气压无关，<br>沸点与大气压有关
section 卡诺
    简介 : 19世纪法国工程师，建立了热力学第二定律
    贡献 : 指出气体作为工质的优势，是热力学的创始人
    著作 : 《关于热动力及热动力机的看法》，<br>热力学发展史的里程碑
section 焦耳
    简介 : 19世纪英国物理学家，奠定了热力学第一定律的实验基础
    贡献 : 给出热功当量值
    著作 : 《论热的机械当量》
```
